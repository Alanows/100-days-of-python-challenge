Project Overview

The Tip Calculator helps users split a restaurant bill evenly among a group of people while including a chosen tip percentage. It calculates how much each person should pay.
How It Works
The program asks the user for the total bill, tip percentage, and number of people splitting the bill.
It calculates the tip amount based on the percentage.
It adds the tip to the bill to get the total.
Finally, it divides the total by the number of people and shows how much each person must pay.
